# Rails-db:resetup

Add rake db:resetup task = db:migrate:reset and db:seed

## Installation

Add this line to your application's Gemfile:

    gem 'rails-db-resetup'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rails-db-resetup

## Usage

    $ rake db:resetup
